# 你画我猜 - 联机游戏

这是一个基于Web的实时多人"你画我猜"游戏，支持局域网联机游戏。

## 功能特性

- 🎨 **实时绘画**: 支持多种颜色和画笔大小
- 💬 **实时聊天**: 玩家可以在聊天框中猜词
- 🏠 **房间系统**: 支持创建和加入房间
- ⏱️ **计时系统**: 每轮限时60秒
- 🏆 **积分系统**: 猜对得分，画家也会得分
- 📱 **响应式设计**: 支持移动端触摸操作

## 游戏规则

1. 每个房间最多支持多名玩家
2. 每轮游戏轮流由一名玩家画图
3. 其他玩家通过聊天框猜测词汇
4. 猜对的玩家获得分数，画家也会获得分数
5. 每轮限时60秒，共进行3轮
6. 游戏结束后显示最终排名

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动服务器
```bash
npm start
```

### 3. 访问游戏
- **本机访问**: `http://localhost:3000` 或 `http://127.0.0.1:3000`
- **局域网访问**: `http://[你的IP地址]:3000`

### 4. 局域网游戏
其他设备可以通过以下地址访问：
```
http://[你的IP地址]:3000
```

例如：`http://192.168.1.100:3000`

服务器会监听所有网络接口 (0.0.0.0:3000)，确保局域网内的设备都能访问。

## 如何游戏

1. **加入游戏**：
   - 输入你的昵称
   - 输入房间号（留空自动生成）
   - 点击"加入游戏"

2. **绘画**（轮到你时）：
   - 选择画笔颜色和大小
   - 在画布上绘制指定词汇
   - 可以使用"清空画布"重新开始

3. **猜词**（其他玩家轮次时）：
   - 观察画家的绘画
   - 在聊天框中输入你的猜测
   - 猜对即可得分

## 技术架构

- **前端**: HTML5 Canvas + CSS3 + 原生JavaScript
- **后端**: Node.js + Express
- **实时通信**: Socket.IO
- **无需数据库**: 游戏状态保存在内存中

## 端口配置

默认端口为3000，你可以通过环境变量修改：
```bash
PORT=8080 npm start
```

## 获取本机IP地址

### Windows
```cmd
ipconfig
```

### macOS/Linux
```bash
ifconfig
# 或者
ip addr show
```

查找类似 `192.168.x.x` 的IP地址，这就是你的局域网IP。

## 故障排除

1. **无法连接**：检查防火墙设置，确保端口3000未被阻止
2. **画布无响应**：确保你是当前轮次的画家
3. **聊天不显示**：检查网络连接状态
